# EduGateway - Education Consultancy Platform

A premium, data-driven education consultancy website built with React and Vite. This platform specializes in providing comprehensive admission guidance for top universities in India and abroad.

## Key Features

- **TN Top Private Colleges**: A curated, premium list of the top 10 private engineering colleges in Tamil Nadu (e.g., VIT, SRM, SSN).
- **Study in Russia**: Detailed listings for 14 top-tier Russian universities across Medical (MBBS) and Engineering streams.
- **Interactive Programs**: Streamlined navigation with a "Top Programs" dropdown and interactive service cards.
- **Global Reach**: Information on medical and engineering programs in Russia, Georgia, Philippines, and Kazakhstan.
- **Lead Capture**: Integrated lead generation forms for streamlined student counseling.

## Tech Stack

- **Frontend**: React.js with Vite
- **Styling**: Vanilla CSS with modern aesthetics
- **Animations**: Framer Motion for premium user interactions
- **Icons**: Lucide React
- **Icons/Graphics**: Custom SVG and rich imagery

## Project Structure

- `src/components/`: Reusable UI components (Navbar, Hero, Services, etc.)
- `src/pages/`: Main page views (Home, RussiaColleges, TamilNaduColleges, etc.)
- `src/data/`: Centralized data store for university and college listings
- `src/App.jsx`: Routing and main application layout

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Start the development server with `npm run dev`
